# 豆腐自制心
最近、日記の更新をサボってしまっていた。原因は自堕落な生活と YouTube 中毒だ。いや、YouTube 中毒になっているからこそ生活が自堕落になっているといったほうが正確かもしれない。

YouTube も夜中しか観られないように制限をかけていたが、5 分置きの cron の監視をその都度外して制限中も何時間と観続けたりしてしまっている。より強固な制限が必要だと思いやり方を模索するがなかなか良い方法が考えつかない。

そしてついに今日の朝 (寝る前) は GitHub の制限がかかっている時間であるにも関わらず YouTube の制限を完全に外してしまった。どうやったかというと、iPhone の Wi-Fi をオフにしてモバイル通信で GitHub アプリを開いて YouTube の制限を外した。いつからか GitHub アプリでも既存ファイルが編集できるようになった。

ここまで来るともう末期だ。最近、本当に誘惑に弱すぎる。なぜここまで誘惑に弱くなってしまったのだろう。そういえば最近瞑想やってなかったな。



# ナイトモード
最近は完全に昼夜逆転している。今日は 19 時に起きた。昨日も 18 時とかに起きた。

そうなると基本的に夜しか起きていないから夜しか洗濯機をかけられない。ということで、少し前から夜に洗濯機をかけるようにし始めた。ただし音が気になるのでナイトモードを使っている。

最近になって初めてナイトモードを使ったのだが、仕上がりの違いなどは正直よくわからない。そこまで洗濯物が汚れるわけではないので。

ただ、洗濯にかかる時間は結構伸びている気がする。やはり音を抑えようとする分、時間がかかるのだろう。

ちなみにルンバも夜遅い時刻にかけるようになった。隣人からうるさいと言われる懸念があるが、今日は珍しく隣に人がいてなんかちょっと物音が気になったのでこっちも別に気にする必要はないかなと。



# プライベート日記執筆前にシークレットモードをオンにする必要はない！
プライベート日記を書くときにシークレットモードをオンにすべし、と以前 (6 日のプライベート日記) に書いたが、めんどくさいし頻繁に忘れてしまう。シークレットモードを忘れずにオンにしたとしても、プライベート日記を書き終えたあとにオフにするのを忘れてしまう。オンにするのとオフにするの、2 回も覚えておかなければならないのもめんどくさい。

しかもシークレットモードがオンだと履歴からの補完が全くできなくなる (そういう目的だから当たり前なんだけど) ので不便だ。別に履歴に残っても良いような単語しか入力しないこともままあるので、これはどちらかというと不便なことのほうが多い。

ここまでしてシークレットモードをオンにする必要ってなんだろうって考えたとき、やはり以前に書いたとおり画面共有などしたときにうっかり見られたくない変換候補 (プライベート日記執筆中に使ったもの) が出てきてしまうのが嫌だから、ということなのだが、そもそも画面共有をすることがほとんどなくなったし、仮に今後することがあったとしても、事前にシークレットモードをオンにすればその問題って解決するんじゃね？ って思った。

それに、Google IME はアカウントとかなくて履歴がクラウドに記憶されるわけでもないから履歴を大切に持っておく必要がない。つまり、どうせパソコンを変えるときに消えてしまうものだから気軽に消せる。そう考えると、度々候補に嫌な単語が出てしまうとかになったら設定から履歴をリセットすれば良いだけの話なのだ。

というわけで、プライベート日記を書く前にわざわざシークレットモードをオンにする必要はないと判断した。



# インクリメンタルリジェネレーションを再度無効に
インクリメンタルリジェネレーションをオンにすると Jekyll のビルドが飛躍的に速くなるのだが、やはりいくつか問題点がある。

* プライベート日記を更新しても (同日の通常日記を更新しないと) 反映されない
* 過去の日記ファイルを新しく作ると其のファイルよりも日付が新しいページの「前の記事」のリンクがおかしくなる
    * まあそもそも時を遡って日記を更新するなって話ではあるんだけど
* 新規ファイルがたまにトラックされない

というわけでまたインクリメンタルリジェネレーションをオフにした。ただしインクリメンタルリジェネレーションのオプションがついたコマンドを削除したわけではなく、またいつでもオンにできるようにコメントアウトした。やはりあのビルドの速さは魅力的だ。

## 追記
…… と思ったけど、毎日 8 時に Jekyll のデーモンを再起動するように cron を更新して、インクリメンタルリジェネレーションを再度有効にした。デーモンの再起動でフルビルドになるので、自動ビルドはインクリメンタルリジェネレーションでも良いかなと思った。
