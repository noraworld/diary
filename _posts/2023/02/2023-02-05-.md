# YouTube の制限を再開
今日の夜、ようやく YouTube のネットワーク制限をかけることに成功した。1 月 1 日から制限をかなり緩くしていたので約 1 ヶ月ぶりの制限である。「成功した」と表現しているのは、やはり誘惑に負けてなかなか制限をかける決断ができなくなるからである。一度制限を外してしまうと、そのあとまた制限をかけるのにものすごくエネルギーが必要になる (誘惑に打ち勝って設定を更新するマインドが必要になる) ので、やはり制限を見直すときは十分に検討すべきだなと感じた。

制限は以下の通りとした。

| 制限前 | 制限後 |
| --- | --- |
| 1 時間半ごとに 30 分の制限 | 00:00 - 05:30 の間で 1 時間半ごとに 30 分の制限 |

制限をかける前 (去年) はたしか 3 日に 1 回観られるという制限だったが、それだとちょっとキツかったのかもしれない。その反動で制限を思いっきり緩くしてしまったのかもしれないなと反省し、夜はお疲れ様的な意味合いも込めて毎日観られるようにした。もちろん毎日観られるから、今日は見ずに英語の勉強をもうちょっとするとかいう日があっても良いかもしれない。

YouTube の制限だけでなく GitHub Editor の制限も一緒に再開した。その夜の視聴できる時間が終了してから GitHub Editor 上で時間をいじることができてしまうと意味がないので。だけど昼間の修正をどうやって緩和すれば良いのかについての答えがまだ出ていない。昼間は仕事とかで GitHub を使うのでこれを制限してしまうと仕事にならなくなってしまう。これは今後の課題である。



# BenQ モニターの再販を開始
今まで BenQ モニターはメルカリ上で非公開にしていたが再開した。値段は 1 円しか変えていないので実質変えていない。以前にもその価格で一度は売れたし、再開してからさっそくコメントがついたので、たぶん放っておけばそのうち売れるだろう。

一度売ったものをなぜまだ持っているのかを日記に書いたかどうか忘れたので改めてここで書いておく。前回売れたとき、大型なので集荷に来てもらったのだが、なんとモニターは台がないと引き取ってもらえないということが判明した。

集荷なしで運ぼうにも 160 cm を超えるのでメルカリ便も使えない。ということは匿名配送ができない。そのうえ、160 cm 以上のモニター用のダンボールだけで Amazon で 5,000 円とかするのでさすがに割に合わないなと思ったのでキャンセルを購入者に要請した。

なので BenQ モニターの重さに耐えられる台を用意してから売ろうと思っていたのだが、なんだか一度こういうことがあってめんどくさくなってしまいそのままクローゼットの上に放置したままにしていた。まだ台は買っていないのだが、売れてからでも間に合うだろうと思いとりあえず出品の再開だけした。

本当は、台も純正のやつしかダメらしいが、台は引っ越しのときに処分しちゃったし台だけ純正をものを手に入れることもできないのでそんなの不可能である。だから嘘をついてでも非純正の台を適当につけて集荷してもらおうという感じになる。

というか台がないと集荷できないとかなんでだろうな。不便すぎる。
