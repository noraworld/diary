# Raspberry Pi のロギングを無効に
ふと、Raspberry Pi 上で systemd のログを保存するのを無効にしたいなと思った。systemd のログってほとんど見ないし何かエラーがあったときもほとんど情報がなくて役に立たない。その上、ディスク I/O が発生するので SD カードの寿命をいたずらに短くするだけだなと思ったのだ。

最近は 1 分おきに cron でネットワーク制御のスクリプトを実行していたりするのでディスク I/O が半端ないんじゃないかなと思ったのだ。事実、cron で何かが実行されるたびに Raspberry Pi の I/O ランプが点滅している気がする。

ということでロギングをオフにすることにした。調べてみるととても簡単に無効にすることができた。

```shell
sudo systemctl stop rsyslog.service
sudo systemctl disable rsyslog.service
```

[Disable all Logging in Ubuntu Server](https://askubuntu.com/questions/861546/disable-all-logging-in-ubuntu-server)

本当に無効にできているかは今後調査する。

```shell
journalctl -xeu dnsmasq.service
```

# Atsueigo School の支払い完了
4 月 5 日の決済を以て Atsueigo School の 12 回分割払いがようやく完了した。[クレジットカードの決済に失敗して](/2023/04/03#クレジットカードを復活させた)いたので最後の支払いが 4 月になってしまったが。

Money Forward で支出を見るたび、この支払いが結構負担に感じていた。結局このスクールは参加する価値があったのかなと振り返ると、残念ながら 710,400 円という大金に見合うサービスではないなと思っていた。それを考えるとスクールが終わったあとも毎月この支払いが発生していることが精神的にも重荷になっていた。しかしそれもようやく終わり、少しだけ心の負担が軽くなった。

スクールが終わったあと、発音スクールがあると聞いたときは正直入るかどうか結構迷ったが結果的に入らなくて良かった。これがまだ続いていたらと思うと、、、。[Atsueigo School メンバーのオフ会に参加](/2023/03/24#atsueigo-school-メンバーオフ会に参加)したときに発音スクールはどうだったか聞いてみたが、やはりただフィードバックがあるだけで何十万も払って参加するほどの価値はなさそうだった。

ちなみにそのときに聞いた話をそのときの日記 (3 月 24 日の日記) に書くのを忘れていたが、その後、発音スクールプラスという名前で、さらに延長で発音スクールが続いているらしい。そしてそれは発音スクール (プラスじゃないほう) に参加していないと応募できないらしくて (プラスがあること自体、オフ会で初めて聞いたし)、今後さらに発音スクールプラスプラスみたいにどんどん延長していくことを考えたときに途中で参加をやめたら次のにも入れない、という恐怖 (？) で惰性で参加させる、みたいな戦略なのかなと思った。個人的にはこういう商売好きじゃないな。ちなみに発音スクールに参加していた 2 人は発音スクールプラスにも参加したらしい。まんまと戦略にハマっているなと思った。もちろん口には出さなかったけど。
